# 🛡️ Cool Defence – Client Website

## 📌 Overview
**Cool Defence** is a responsive business website developed for **Digambara Industries**, a manufacturer of Diesel Exhaust Fluid (DEF) in Pune. The site showcases products, 
certifications, and company details with a clean and modern UI.

---

## 🚀 Features

* Multi-page website (Home, About, Products, Certifications, Contact)
* Fully responsive design
* Product showcase (DEF & Coolant)
* Certifications display (ISO standards)
* Contact section with Google Maps & WhatsApp integration
* Scroll animations using AOS

---

## 🛠️ Tech Stack

* HTML, CSS, JavaScript
* Bootstrap 5
* AOS Animation Library
